---
name: "agent-handoff-planning"
description: "Structure a clean handoff between workflow agents with explicit outputs, open questions, and next-owner guidance."
---

When one workflow agent hands work to another:

- summarize what is already decided
- separate confirmed facts from assumptions
- list unresolved questions
- name the exact next owner
- state the expected deliverable from that next owner

Avoid vague handoffs like "someone should look at this next."
